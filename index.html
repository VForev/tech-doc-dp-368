Python Syntax

Python uses indentation to define code blocks...

Here's an example of a simple condition...

if x > 5: print("x is greater than 5") def greet(name): return f"Hello, {name}" print(greet("Python"))
  • Indentation is required for blocks
  • Use colons (:) to define conditions
  • Function definition uses the 'def' keyword
Python Data Types

Python supports various data types...

Common data types in Python include:

x = 10 # Integer y = "Hello" # String z = [1, 2, 3] # List t = (4, 5, 6) # Tuple d = {"name": "Python", "type": "programming"} # Dictionary
  • Integer: Whole numbers
  • String: Textual data
  • List: Ordered collection of items
  • Tuple: Immutable collection
  • Dictionary: Key-value pairs
Functions in Python

Functions are defined using the def keyword...

Here's an example of a simple function:

def greet(name): print(f"Hello, {name}")
  • Functions use the 'def' keyword
  • Indentation is important in defining the body
  • Functions can return values
Python Libraries

Python has a wide range of libraries...

Libraries can be imported using the import keyword.

import math import datetime import os import sys import requests
  • math: Provides mathematical functions
  • datetime: For working with dates and times
  • os: For interacting with the operating system
  • sys: Provides access to system-specific parameters
  • requests: For sending HTTP requests
